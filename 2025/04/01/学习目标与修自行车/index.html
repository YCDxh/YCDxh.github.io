1.Java 语言的特点（如果你简历上有提到 C++ 可能还会问你 Java 和 C++ 的区别）。【⭐⭐】

面向对象
封装 继承 多态

  • 多态就是说子类可以重写父类的方法，

2.比较 JVM 和 JDK 以及 JRE 。【⭐⭐】

  1. JVM 是运行 Java 字节码的虚拟机, 是 Java 语言“一次编译，随处可以运行”的关键所在
  2. JDK 是一个功能齐全的 Java 开发工具包，供开发者使用
  3. JRE 是运行已编译 Java 程序所需的环境, 主要包括 JVM 和 Java 基础类库

简单来说，JRE 只包含运行 Java 程序所需的环境和类库，而 JDK 不仅包含 JRE，还包括用于开发和调试 Java 程序的工具。

1. Integrated and DC-powered superconducting microcomb

Nature Communications 15, 4009 (2024) （Nature子刊）

频率梳

一种发射多个等间距频率线的特殊激光源，以其前所未有的精确度和多功能性彻底改变了科学和技术。

最近，集成频率梳正作为片上光子学的可扩展解决方案而崭露头角。

在这里，我们展示了一种完全集成的超导微梳，它易于制造、操作简单，并且功耗极低。

我们的成套设备包括一个基本的非线性超导器件——约瑟夫森结，直接耦合到一个超导微带谐振器。

我们展示了通过自启动模式锁定生成相干频率梳的技术。

因此，梳状辐射仅需激活直流偏压源即可开始，功率消耗低至数十皮瓦特。产生的梳状光谱位于微波领域，并跨越多个八度音阶。

所有梳状线的线宽可以通过独特的相干注入锁定技术缩小到1赫兹。我们的工作是迈向完全集成微波光子学的重要一步，并为集成量子处理器提供了潜力。

2. Toroidic phase transitions in a direct-kagome artificial spin ice

Nature Nanotechnology, 1-7 (2024) （Nature子刊）

人工自旋冰

论文摘要：

铁环状性（铁涡性），作为预测为初级铁磁序的第四种形式，打破了空间和时间反演对称性，并代表了技术进步的一个引人注目的途径。

然而，在天然材料中实现铁环状性已被证明具有挑战性，这阻碍了对铁环状相变的探索。

在这里，我们通过在人工纳米磁体系统中探索铁环状性克服了天然材料的局限性，该系统可以在组成水平上进行表征，并且可以在不同的有效温度下进行热退火处理。

我们引入了一种新的纳米磁体阵列——直接卡格姆自旋冰的第一实现。

这种独特的人工自旋冰展示了稳健的环形矩和准简并基态，导致两个不同的低温环形相：铁环状性和偏环状性。

我们在实验上和数值上展示了铁环状性和偏环状性之间的相变，以及向非环形顺磁相的交叉。

我们的准简并直接卡格姆结构的人工自旋冰提供了一个新模型系统，用于研究在天然材料中无法获得的磁状态和相变。

  • 约瑟夫森结——三明治超导，高度非线性。

    • 作为”电压-频率转换器”，将直流电压转换为微波频率

    • 产生非线性效应，这是形成频率梳的关键（不懂）

    • 提供能量来源（仅需40pW的直流功率）

  • 超导微带谐振器——微波传输线结构

    • 作为高品质因数的谐振腔，存储和增强微波光子

    • 提供多模谐振条件（自由光谱范围0.75GHz）

    • 与约瑟夫森结强耦合，实现自启动模式锁定

  • 谐振腔：微波回音室，起储存特定频率电磁波的作用

  • 微波光子：电磁波的能量量子化单位，超导电路中表现为谐振腔的电磁场振动

  • 耦合：两个器件之间的能量交换机制

  • 相位相干：各频率分量保持固定的相位关系，时域上形成稳定脉冲序列

  • 多模谐振：谐振腔同时支持多个谐振模式（不懂）

耦合（Coupling）
指的是两个或多个系统、组件或物理量之间
通过某种机制发生相互作用，导致能量、信息或状态的相互传递或影响。

耦合是自然界和技术系统中广泛存在的现象，其形式和强度直接影响系统的整体行为。

耦合的核心作用是建立不同组件之间的连接通道，使能量（如电磁能、机械能）或信息（如信号、量子态）能够从一个系统传递到另一个系统。

  • 电磁耦合：通过电场或磁场相互作用（如天线间的电磁波耦合）。

  • 机械耦合：通过机械振动或应力传递（如微机电系统（MEMS）中的悬臂梁耦合）。

  • 直接耦合：物理连接（如导线直接连接电路元件）。

  • 间接耦合：通过中间媒介（如光通过光纤耦合到探测器）。

文章提到在微波域内工作的完全集成频率梳仍然难以实现 ？为什么

能量转换效率问题

  • 传统方案需要将光学频率梳下转换到微波域，这会导致：

    • 转换效率低（典型值＜10%）
    • 引入额外噪声
    • 系统复杂度急剧增加（需要光学泵浦、非线性晶体等）

  • 微波光子寿命限制微波谐振腔的光子存储时间（τ∝Q/f）受限于：

    • 材料损耗（超导材料表面阻抗）
    • 辐射损耗（特别是在高频段）
    • 典型值：~μs量级（比光学谐振腔低3个数量级）

  • 非线性效应不足，传统微波器件（如半导体器件）的非线性系数：

    • χ⁽³⁾~10⁻¹⁸ m²/W（比光学非线性材料低6个量级）
    • 导致需要极高的泵浦功率（通常＞1W）

  • 集成工艺挑战

    • 超导电路（需低温）与半导体器件的兼容性
    • 微波-光波混合集成时的阻抗匹配问题
    • 典型加工误差（100nm）对微波器件影响更大（λcm-mm）

  • 噪声控制困境，微波域更易受：

    • 1/f噪声（低频噪声主导）
    • 热涨落噪声（kBT≈4μW/GHz at 4K）
    • 相位噪声积累（因品质因数Q较低）

本文的创新突破点在于：

  • 直接利用约瑟夫森结的固有非线性（无需光学转换）

  • 超导谐振腔实现超高Q值（＞10⁶ @4K）

  • 直流驱动方案（功耗仅40pW）

  • 全超导单片集成（避免异质集成问题）
